Welcome to Persephone Web API's documentation!
==============================================

The Persephone Web API provides a `REST <https://en.wikipedia.org/wiki/Representational_state_transfer>`_ API to interact with the `Persephone automatic phoneme transcription library <https://github.com/persephone-tools/persephone>`_

The `Persephone Web Frontend <https://github.com/persephone-tools/persephone-frontend>`_ is an example this API being used.
